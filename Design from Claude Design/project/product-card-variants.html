01

Lot header

Catalog number on the left in mono caps. On the right, a status pip in Wax Seal — Available / Reserved / Sold. The horizontal hairline below is the only place the frame's full 0.5px solid Bark Black repeats inside the card.

02

Stage

Image is set on a Raw Linen plinth (24px padding all around) — never edge-to-edge. The plinth is the pedestal; the work sits on it.

03

Title block

Roman numeral in Wax Seal (Cormorant 300 · 36px) reads like a sonnet number. Title in Cormorant 400 · 26px sits next to it on the baseline, allowing the work's name to wrap freely.

04

Description

Italic serif (Cormorant 300 italic · 14px), justified. One short paragraph — never more. Reads like a catalog entry, not marketing.

05

Properties

Tabular metadata in JetBrains Mono 10px / 0.18em tracking. Labels in Smoked Oak, values in Bark Black. The Latin binomial for the branch source is required — provenance is the brand.

06

Estimate

"Estimate" label in mono caps; price in Cormorant 500 · 24px. Separated from the body by a full 0.5px solid Bark Black rule. We never write "Price" — this is a lot.

07

Foot stamp

Wax-seal pip + Sealed · signed verso on the left. The CTA — "Bring it home" — is a hairline-underlined text link in mono caps, never a filled button. The card itself is the invitation.

08

Frame

The whole card is enclosed in a single 0.5px solid Bark Black hairline. No shadow, no rounding. Width is fluid; height grows with content. Min width 320px, recommended 360–440px.
